Latar Belakang & Masalah

Akses Terfragmentasi

Informasi beasiswa tersebar di berbagai platform, membuat mahasiswa kesulitan mencari data yang valid.

Manajemen Data (Sudut Pandang Hacker)

Belum adanya sistem database terpusat yang efisien untuk menyajikan informasi beasiswa secara terstruktur dan real-time.

Target User Persona

👩‍🎓

Reva

Siswi SMK Kelas 12

Goal: Mendapatkan beasiswa untuk mengubah masa depan keluarganya serta mencari informasi beasiswa yang jelas, valid, dan terarah.

Pain Point: Informasi beasiswa tersebar di banyak platform. Deadline pendaftaran sering mepet atau terlewat.

Behavior: Aktif mencari informasi di Instagram, TikTok, website kampus, dan Telegram.

Insight Hasil Riset

Pentingnya Notifikasi

80% Responden menginginkan pengingat otomatis 7 hari sebelum penutupan pendaftaran.

Filter Cerdas

Pengguna membutuhkan fitur pencarian dan filter klasifikasi berdasarkan nilai IPK, Jurusan, serta Jenis Beasiswa (Prestasi/Bantuan).

Validasi Data

Keamanan dan keaslian info beasiswa menjadi prioritas utama pengguna agar terhindar dari modus penipuan.

Solusi & Fokus Pengembangan

Aplikasi CaBe (Cari Beasiswa) dibangun menggunakan Flutter Framework & Bahasa Dart dengan arsitektur bersih, memastikan performa antarmuka yang responsif, ringan, serta konsisten di berbagai perangkat mobile.

  • Smart Dashboard: Implementasi manajemen state yang efisien untuk menyajikan informasi beasiswa terbaru langsung di layar utama.
  • Scholarship Matching: Struktur logika algoritma terintegrasi untuk memetakan kualifikasi pengguna dengan syarat kriteria beasiswa secara real-time.
  • Local Data Persistence: Sistem penyimpanan internal lokal yang aman untuk menunjang fitur simpan beasiswa favorit (One-Click Save).

Core Implementation (Hacker Role)

Berikut merupakan potongan arsitektur logika program untuk inisialisasi transisi animasi halaman pembuka aplikasi (Splash Screen) menggunakan Flutter:

import 'package:flutter/material.dart';
import 'dart:async';
import '../theme/app_colors.dart';
import 'home_screen.dart';

class SplashScreen extends StatefulWidget {
  const SplashScreen({super.key});

  @override
  State createState() => _SplashScreenState();
}

class _SplashScreenState extends State with SingleTickerProviderStateMixin {
  late AnimationController _controller;
  late Animation _colorAnimation;

  @override
  void initState() {
    super.main.initState();
    _controller = AnimationController(
      duration: const Duration(seconds: 5),
      vsync: this,
    );

    _colorAnimation = ColorTween(
      begin: AppColors.blue200,
      end: AppColors.white,
    ).animate(CurvedAnimation(parent: _controller, curve: Curves.easeInOut));

    _controller.forward();

    Timer(const Duration(seconds: 3), () {
      if (mounted) {
        Navigator.of(context).pushReplacement(
          MaterialPageRoute(builder: (context) => const HomePage()),
        );
      }
    });
  }
}
